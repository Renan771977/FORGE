Isso define o equilíbrio entre CPU e GPU na build.

Máxima GPU

Máxima CPU

Equilibrado

Mix de tarefas. Nenhum componente é gargalo.

Alta RAM + Storage

Muitos processos abertos, arquivos grandes, edição RAW.
